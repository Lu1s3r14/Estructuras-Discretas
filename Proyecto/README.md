# Proyecto de Laboratorio de Estructuras Discretas.
### Objetivos:
1. Aprender sobre la codificacion de Huffman
2. Saber implementar la variante de codificacion
3. Aprender mejor sobre el uso de LaTex
4. Conocer para que es y su uso de la compresion de datos
5. Saber la diferencia entre la variante de la codificacion y la tradicional.

## Comentarios acerca del proyecto
Apesar de que es algo relativamente nuevo para mi, debido a que en algun momento si habia escuchado acerca de la codificacion de Huffman siento que la implementacion no fue lo mas complicado a la hora de trabajar, sino hacer el reporte usando LaTex ya que solo habia realizado algo sencillo en el propedeutico y cuando vi el archivo del proyecyo si me asombre bastante acerca del uso mas avanzado de LaTex, siento que eso fue lo que mas tiempo me llevo.
